# 防蓝光护眼助手 BlueLight Filter Pro

A powerful Chrome extension designed to protect your eyes by filtering blue light and reducing eye strain during computer use.
一款强大的Chrome扩展，通过过滤蓝光来保护眼睛健康，减少长时间使用电脑带来的眼疲劳。

## Features | 功能特点

- Advanced blue light filtering technology | 先进的蓝光过滤技术
- Customizable filter intensity | 可自定义过滤强度
- Automatic scheduling options | 自动定时功能
- Eye-friendly interface | 护眼友好的界面
- Real-time protection | 实时保护
- Works across all websites | 适用于所有网站

## Installation | 安装

1. Download the extension from the Chrome Web Store | 从Chrome网上应用店下载扩展
2. Click "Add to Chrome" to install | 点击"添加到Chrome"进行安装
3. The extension icon will appear in your Chrome toolbar | 扩展图标将出现在Chrome工具栏中

## Usage | 使用方法

1. Click the extension icon in your Chrome toolbar | 点击Chrome工具栏中的扩展图标
2. Adjust the filter intensity using the slider | 使用滑块调整过滤强度
3. Set up automatic scheduling if desired | 根据需要设置自动定时
4. Enjoy reduced eye strain while browsing | 享受浏览时减少的眼疲劳

## Benefits | 优势

- Reduces eye strain and fatigue | 减少眼疲劳
- Improves sleep quality | 改善睡眠质量
- Protects long-term eye health | 保护长期眼部健康
- Customizable to your needs | 可根据需求自定义
- Works seamlessly in the background | 在后台无缝运行
